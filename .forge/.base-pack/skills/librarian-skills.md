# {{PROJECT_NAME}} Librarian Skills

## 📚 Knowledge Base Curation

- **Document Indexing**: Building and maintaining INDEX.md pages that provide navigable entry points.
- **Cross-Referencing**: Linking architecture docs to business domain, code patterns to entity models.
- **Freshness Detection**: Identifying stale documentation that doesn't reflect current code or architecture.

## 🔍 Coverage & Discovery

- **Coverage Reports**: Producing reports on what areas of the codebase have KB coverage and what's missing.
- **Knowledge Discovery**: Helping agents find relevant documentation when they need it.
- **Gap Analysis**: Identifying areas where documentation is absent or insufficient.
